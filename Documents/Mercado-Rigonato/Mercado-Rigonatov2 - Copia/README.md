# Mercado Rigonato - Sistema de E-commerce

## Estrutura de Imagens

### Banners
- Coloque os banners em `img/banner/`
- Nomeie como: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`

### Produtos
- Organize por categorias em `img/produtos/`
- Exemplo:
  - `img/produtos/acougue/picanha.jpg`
  - `img/produtos/padaria/pao-frances.jpg`
  - `img/produtos/hortifruti/tomate.jpg`

### Logo
- Coloque a logo em `img/logo.png`

## Como Adicionar Imagens

1. **Banners:** Substitua os arquivos em `img/banner/`
2. **Produtos:** Adicione nas pastas correspondentes à categoria
3. **Logo:** Substitua `img/logo.png`

O sistema está configurado para usar essa estrutura automaticamente!
